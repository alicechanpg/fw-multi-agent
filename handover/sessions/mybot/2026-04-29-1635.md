### 重要陷阱（避免重蹈覆轍）
- **VS2017 lambda capture 不認 forward-declared pointer type** — 用 bool 容器規避
- **Jenkins ChoiceParameter 嚴格 enforce list** — 不在 list 的 branch 會 HTTP 500，要先改 config.xml POST 加 branch
- **`git submodule update` 會把 working tree reset 到 parent 的 pointer** — multi-stage push 中間別跑這個
- **Spark MINI Actions chip 跟 GD32 是分離的** — J-Link 只能 reset GD32，Actions chip 要物理斷電
- **`juce::Timer` 永遠在 message thread 跑** — 任何 inherit Timer 的 class 內 timerCallback 都不能直接做 blocking MIDI / I/O，必須 dispatch 到 worker thread
