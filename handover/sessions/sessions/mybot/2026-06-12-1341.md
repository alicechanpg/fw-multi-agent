## Manual Run

```powershell
cd D:\mybot\git\tool
python ble_observer.py --cycles 1 --no-capture --timeout 600
```

This will:
1. Open STM32 serial (COM4, 921600) — monitor OTA progress + reboot
2. BLE connect to `98:A3:16:DC:F4:52` — subscribe 0xFFC2 notifications
3. Wait for ESP32 simulation mode to trigger OTA
4. STM32 serial shows "Bootloader" → OTA cycle done
5. Print verdict: PASS (got FINISH) / FAIL (no FINISH or error state)

**Important**: Don't open COM7 during OTA — it causes ESP32 USB_UART_CHIP_RESET!
