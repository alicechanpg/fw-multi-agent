---
name: No Org Chart in JIRA
description: Never write organizational hierarchy or reporting structure into JIRA tickets
type: feedback
---

組織架構（誰的老闆是誰、reporting line）不可以寫到 JIRA ticket 裡。

**Why:** 組織架構是內部敏感資訊，不適合出現在 JIRA 這種多人可見的系統上。

**How to apply:** 組織架構只存在 memory 裡（local），不放進任何外部系統（JIRA、Slack、Google Docs、GitHub）。
